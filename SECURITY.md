# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability
Please report vulnerabilities via:
1. GitHub Issues (for public)
2. Email (for sensitive issues)

## Response Time
- Critical: 24 hours
- High: 48 hours
- Medium: 1 week
